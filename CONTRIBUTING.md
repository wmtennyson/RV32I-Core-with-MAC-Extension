# Contributing to Senior Project - 2025

Thank you for considering contributing to this project!  
Here’s how you can help:

## How to Contribute

1. **Fork** the repo and clone your fork.
2. **Create a new branch** for your feature or fix.
3. **Commit** your changes with clear, descriptive messages.
4. **Push** to your fork and open a **Pull Request** on the main repo.

## Code Guidelines

- Use clear, consistent naming for files and modules.
- Document your code with comments.
- Add or update testbenches for any new modules.

## Issues & Discussions

- Use GitHub Issues to report bugs or suggest features.
- Use Discussions for design questions or brainstorming.

## Code Reviews

All contributions will be reviewed by a teammate before merging.

## Need Help?

Open an issue or ask in the Discussions tab!
