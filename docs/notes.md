Project Idea: Build a 32-Bit CPU

Reason: Oriented for Digital Signal Processing

Instruction Set Architecture (ISA): RISC-V 

Hardware Description Language (HDL): Verilog (90%), SystemVerilog (10%)

What to Buy: 
FPGA Board 

Components: 
- ALU
- DSP Unit 
- Register File
- Program Counter
- Instruction Memory
- Data Memory
- Memory Instruction Set
- Controller (Control Unit)
